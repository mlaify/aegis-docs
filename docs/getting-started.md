# Getting Started

This guide gives the fastest path to a local Aegis v0.1 development loop.

## Step 1: Read the Protocol Index

Start with:

- `../aegis-spec/docs/protocol-index.md`

That index points to the active RFC set and current guarantee boundaries.

## Step 2: Run the Local E2E Demo

Use the CLI demo script from `aegit-cli`:

```sh
sh ../aegit-cli/scripts/local-e2e-demo.sh
```

See full instructions:

- `../aegit-cli/DEV-SETUP.md`

## Step 3: Understand Conformance

Read:

- `../aegis-spec/docs/implementation-conformance-v0.1.md`

This maps RFC requirements to current code.

## v0.1 Reminder

Aegis is draft/prototype and demo crypto/signing is non-production.
